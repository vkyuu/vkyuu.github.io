Research

After Assets and Performance review of the current website done by Rack strategic team, we've found that:

  • Users hardly find value between each products.
  • Navigations are confusing
  • Lack of users engagement
  • Too many steps before reaching leads generation

Users already know what they're looking for before visiting the website which means user's intentions has already been established

Strategy

We provide these strategies as solutions:

  • Start serving instead of selling
  • Create solutions rather than multiply features
  • Think of the product as a connected flow that the user takes

Our proposed UI/UX Directions:

  • Clean and dynamics visual to improve ease of use
  • Create feature that utilized as interactive navigation
  • Fresh and humanist interaction to capture the feeling of refreshment
